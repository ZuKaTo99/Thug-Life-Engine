# Thug Life Engine V_0.1

Web-Wallpaper für Wallpaper Engine.

## Stand

- Originalbild bleibt unverändert und wird als Hintergrund verwendet.
- Zielausrichtung: 2560 x 1440, funktioniert aber responsive auch auf anderen 16:9-Auflösungen.
- Animierte Wolken oben.
- Automatischer Tag-/Nachtwechsel über einen 20-Minuten-Zyklus.
- Dezente Atmosphäre mit Sternen, Staubpartikeln, Licht, Haze und Vignette.
- Kleines Mini-Spiel unten: Figur springt per Klick/Leertaste und sammelt Münzen.
- Münzen werden lokal im Browser/Wallpaper gespeichert.
